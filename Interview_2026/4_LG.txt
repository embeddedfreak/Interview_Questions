Problem Statement

There are N boxes, each containing a prime number. A sequence is generated by taking the multiples of the prime numbers round by round.

In the first round, take the 1st multiple of every prime number.
In the second round, take the 2nd multiple of every prime number.
In the third round, take the 3rd multiple of every prime number.
Continue this process indefinitely.

For example, if the prime numbers are 3, 5, 7, the generated sequence is:

3, 5, 7, 6, 10, 14, 9, 15, 21, 12, 20, 28, ...

Given N, M, and the list of N prime numbers, write a program to print the M-th element in the generated sequence.

Input Format
The first line contains two integers N and M.
The second line contains N space-separated prime numbers.
Output Format

Print the M-th element in the generated sequence.

Constraints
1 ≤ N ≤ 10^5
1 ≤ M ≤ 10^9
All given numbers are prime.
Example 1

Input

1 3
7


Generated Sequence

7, 14, 21, 28, ...


Output

21


Explanation

There is only one prime number, 7. Its multiples form the sequence 7, 14, 21, .... The 3rd element is 21.

Example 2

Input

3 3
3 5 7


Generated Sequence

3, 5, 7, 6, 10, 14, 9, 15, 21, ...


Output

7


Explanation

The sequence is formed by taking the multiples of all prime numbers level by level:

1st multiples: 3, 5, 7
2nd multiples: 6, 10, 14
3rd multiples: 9, 15, 21
...

The 3rd element in the overall sequence is 7.

Problem Statement

Given an array of N integers and an integer K, consider every contiguous subarray (window) of size K.

For each window, determine the difference between its largest and smallest elements. Among all such windows, print the smallest difference obtained.

Input Format
The first line contains an integer N, representing the size of the array.
The second line contains N space-separated integers.
The third line contains an integer K, representing the window size.
Output Format

Print a single integer representing the minimum difference obtained among all windows of size K.

Example 1

Input

6
1 3 2 6 4 5
3


Output

2


Explanation

The windows of size 3 are:

Window	Difference
[1, 3, 2]	3 - 1 = 2
[3, 2, 6]	6 - 2 = 4
[2, 6, 4]	6 - 2 = 4
[6, 4, 5]	6 - 4 = 2

The smallest difference is 2.

Example 2

Input

7
8 2 4 7 9 1 5
2


Output

2


Explanation

The windows of size 2 are:

Window	Difference
[8, 2]	8 - 2 = 6
[2, 4]	4 - 2 = 2
[4, 7]	7 - 4 = 3
[7, 9]	9 - 7 = 2
[9, 1]	9 - 1 = 8
[1, 5]	5 - 1 = 4

The smallest difference is 2.

This is a natural way such questions are presented in coding tests: the problem statement describes the task, while the examples demonstrate exactly how the "difference" is computed.
